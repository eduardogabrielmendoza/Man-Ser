# Man-Ser

About the Project

Man-Ser is a professional website for an industrial machinery company specializing in controlled chip management within the industry. This project showcases modern web development practices with a focus on user experience:

- 5 complete pages with full navigation (Home, About, Products, Services, Contact)
- Mobile-First design that is fully responsive
- Smooth animations and interactive components
- Accordion/Modal system for content display
- Image sliders and galleries
- Professional industrial design aesthetic
- Contact forms and interactive elements

Technology Stack

- Server: Node.js with Express
- Markup: HTML5
- Styles: Custom CSS with modular architecture
- Animations: Custom CSS animations and transitions
- Typography: Inter font family
- Color Palette: Primary (#1a4d8c), Secondary (#f39c12), Neutral grays

Features

Interactive Components
- Accordion system for desktop with modal popups for mobile
- Image sliders with smooth transitions
- Hover effects on cards and buttons
- Interactive navigation with backdrop blur

Product Pages
- Washing machines with detailed specifications
- Telescopic protectors for machinery
- Chip extractors and conveyor belts
- Multiple configuration options and technical details

Service Pages
- Plasma cutting services
- CNC punching and sheet metal bending
- Mechanical machining
- Welding and assembly
- Design and manufacturing adaptations

Responsive Design
- Mobile-first approach with breakpoints at 480px, 768px, 1024px, 1440px
- Touch-optimized interactions
- Adaptive layouts for all screen sizes
- No horizontal scrolling on mobile devices

Performance Optimizations
- Lazy loading for images
- Optimized CSS with minimal specificity
- Smooth 60fps animations
- Efficient DOM manipulation
